#GIPS
The [GIPS](http://github.com/gipit/gips) project is an open-source library and collection of command line utilties to support the retrieval, management, inventorying, and processing of geospatial raster data.

####Supported Drivers

* Landsat 5,7,8
* MODIS

####Utilities

* gips_inventory
* gips_process
* gips_project
